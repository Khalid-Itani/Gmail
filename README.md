# Gmail (CodepathMail)

A simple Gmail-style email list built for a CodePath Android lab. This app focuses on implementing scrolling lists using `RecyclerView`, custom row layouts, and basic UI behaviors.

## Features

### Required
- Displays a scrollable list of 10 emails
- Each email shows:
  - Sender
  - Title
  - Summary

### Stretch
- Load More button appends 5 additional emails to the list
- Each email also shows:
  - Sender icon
  - Date
- Unread emails are bold until tapped (tap marks the email as read)
